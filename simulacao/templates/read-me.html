Manual de utilização da API - Simulação

[SEU HOST]/get-taxa | [GET]
Descrição

Rota responsavel de buscar e retornar segundo paramentros a taxa de juros selecionada.

Paramentros
  • tipo (Obrigatório) : String (enum) ["NEGATIVADO", "SCORE_ALTO","SCORE_BAIXO"]
  • numeroParcelas (Obrigatório) : int (enum) [6, 12, 18, 24, 36]
Retorno
  • code : 400:Bad Request
    { result:{ "error":{ "msg": "400 error: Requisição inválida.", "errors": [DESCRIÇÃO DO ERRO DETECTADO] } } }
  • code : 403:forbidden
    { result:{ "error":{ "msg": "403 error: O registro solicitado não existe." } } }
  • code : 200:success(200)
    { result:[TAXAJUROS] }
[SEU HOST]/calculo | [POST]
Descrição

Rota responsavel calcular com base nos parametros enviados, o valor do montante e do valor por parcela da simulação de emprestimo. Utilizando a formula de juros compostos.

Paramentros
  • valor (Obrigatório) : float | int -> [valor] > 0
  • numeroParcelas (Obrigatório) : int (enum) [6, 12, 18, 24, 36]
  • taxaJuros (Obrigatório) : float -> [taxaJuros] < 1
Retorno
  • code : 400:Bad Request
    { result:{ "error":{ "msg": "400 error: Requisição inválida.", "errors": [DESCRIÇÃO DO ERRO DETECTADO] } } }
  • code : 200:success(200)
    { "result": { "total": [VALOR_TOTAL], "total_parcela": [VALOR_POR_PARCELA] } }
