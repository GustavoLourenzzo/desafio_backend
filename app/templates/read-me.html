Manual de utilização da API - Geral

[SEU HOST]/simulacao | [POST]
Descrição

Rota responsável por validar os paramentros necessarios e acionar a API responsavel para calcular o valor simulado.

Paramentros
  • valor (Obrigatório) : float | int -> [valor] > 0
  • numeroParcelas (Obrigatório) : int (enum) [6, 12, 18, 24, 36]
  • user : Array -> null | Object ModelUser ({ "celular": [CELULAR], "cpf": [CPF], "negativado": [NEGATIVADO], "nome": [NOME], "score": [SCORE] })
Retorno
  • code : 400:Bad Request
    { result:{ "error":{ "msg": "400 error: Requisição inválida.", "errors": [DESCRIÇÃO DO ERRO DETECTADO] } } }
  • code : 403:forbidden
    { result:{ "error":{ "msg": "403 error: O registro solicitado não existe." } } }
  • code : 500: Internal Server Error
    { result:{ "error":{ "msg": "500 error: Houve um problema inesperado com o servidor." } } }
  • code : 200:success(200)
    {result: {qtd_parcelas: [NUMERO_PARCELAS], taxa: [TAXA_JUROS], total: [VALOR_TOTAL], total_parcela: [VALOR_PARCELA], v_inicial: [VALOR_INICIAL]}}
[SEU HOST]/login | [POST]
Descrição

Rota responsavel acionar a api de identificação do usuario e caso o registro seja encontrado retornar o registro.

Paramentros
  • cpf (Obrigatório) : String -> min = max = 11
  • celular (Obrigatório) : String -> min =10, ma=11
Retorno
  • code : 400:Bad Request
    { result:{ "error":{ "msg": "400 error: Requisição inválida.", "errors": [DESCRIÇÃO DO ERRO DETECTADO] } } }
  • code : 403:forbidden
    { result:{ "error":{ "msg": "403 error: O registro solicitado não existe." } } }
  • code : 200:success(200)
    { "result": { "celular": [CELULAR], "cpf": [CPF], "negativado": [NEGATIVADO], "nome": [NOME], "score": [SCORE] } }
[SEU HOST]/usuario/cadastrar | [POST]
Descrição

Rota responsavel por acionar a api de identificação do usuario e realizar o cadastro do registro.

Paramentros
  • cpf (Obrigatório) : String -> min = max = 11
  • celular (Obrigatório) : String -> min =10, ma=11
  • nome (Obrigatório) : String -> min = 4
  • score (Obrigatório) : int -> min = 0, max = 1000
  • negativado (Obrigatório) : bool
Retorno
  • code : 400:Bad Request
    { result:{ "error":{ "msg": "400 error: Requisição inválida.", "errors": [DESCRIÇÃO DO ERRO DETECTADO] } } }
  • code : 201:success(201)
    { {'result': "Registro gravado com sucesso."} }
