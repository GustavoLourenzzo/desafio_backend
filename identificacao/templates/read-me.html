Manual de utilização da API - Identificação

[SEU HOST]/get-user | [GET]
Descrição

Rota responsavel por buscar um usuario na base identificados pelo par cpf - celular.

Paramentros
  • cpf (Obrigatório) : String -> min = max = 11
  • celular (Obrigatório) : String -> min =10, ma=11
Retorno
  • code : 400:Bad Request
    { result:{ "error":{ "msg": "400 error: Requisição inválida.", "errors": [DESCRIÇÃO DO ERRO DETECTADO] } } }
  • code : 403:forbidden
    { result:{ "error":{ "msg": "403 error: O registro solicitado não existe." } } }
  • code : 200:success(200)
    { "result": { "celular": [CELULAR], "cpf": [CPF], "negativado": [NEGATIVADO], "nome": [NOME], "score": [SCORE] } }
[SEU HOST]/cadastra | [POST]
Descrição

Rota responsavel por inserir novos usuarios na base

Paramentros
  • cpf (Obrigatório) : String -> min = max = 11
  • celular (Obrigatório) : String -> min =10, ma=11
  • nome (Obrigatório) : String -> min = 4
  • score (Obrigatório) : int -> min = 0, max = 1000
  • negativado (Obrigatório) : bool
Retorno
  • code : 400:Bad Request
    { result:{ "error":{ "msg": "400 error: Requisição inválida.", "errors": [DESCRIÇÃO DO ERRO DETECTADO] } } }
  • code : 201:success(201)
    { {'result': "Registro gravado com sucesso."} }
